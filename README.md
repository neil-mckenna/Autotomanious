## Unity AI Guard System

A Unity project demonstrating AI systems, An Experiment

Features

FSM and BT AI

Player controller Stealth mechanics

## Created by Neil Mckenna

## How to Run

1. **Install Unity 6** from unity.com
2. **Clone this repository** or download ZIP
3. **Open Unity Hub** → Add → Select project folder
4. **Wait for import** (first time takes a few minutes)
5. **Open a scene** (Assets/Scenes/FSM_Level.unity)
6. **Press Play** to test in Editor

## Building an Executable

1. File → Build Settings
2. Add scenes
3. Select platform
4. Click Build
